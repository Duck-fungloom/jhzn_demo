# DESIGN.md - 认知伙伴 (Cognitive Companion)

## 气质与意象

**关键词**: 温暖陪伴、信任感、成长路径、情绪安全

**意象场景**: 深夜自习室里一盏暖色台灯，桌上摊开的笔记本旁边放着一杯温热的茶。窗外是安静的校园，但这里让人感到安全和被理解。不是冰冷的AI工具，而是一个"记得你"的学长/学姐，在你最焦虑的时候说"我懂，我陪你走"。

## 视觉策略

- **摄影/图像方向**: 使用柔和的抽象渐变作为装饰，避免生硬的几何图形
- **图形语言**: 圆角卡片 + 时间线连接线 + 呼吸光晕动画，模拟一条"走过的路"
- **图标**: 使用 Feather 图标集，线条圆润，2px 线宽
- **数据可视化**: 柔和柱状图，圆角柱体，渐变色填充

## 配色方案

基于柔和卡片风，但调整为更具"治愈感"的色调：

| 角色 | 色值 | 意象来源 |
|------|------|----------|
| 主色 | #6C63FF (薰衣草紫) | 信任、平静、智慧 |
| 主色渐变 | #896BFF | 温暖过渡 |
| 辅色 | #FF6584 (珊瑚粉) | 关怀、温暖、情感支持 |
| 背景 | #F0F0F3 (暖灰白) | 不刺眼的底色，像纸张 |
| 凹陷面 | #E8E8EB | 输入区域、内嵌区域 |
| 高光阴影 | #FFFFFF | 新拟态凸起效果 |
| 暗部阴影 | #D1D9E6 | 新拟态凹陷效果 |
| 文字主 | #2D3436 | 柔和的深色，不用纯黑 |
| 文字次 | #636E72 | 辅助信息 |
| 成功色 | #00B894 (薄荷绿) | 进步、完成 |
| 警告色 | #FDCB6E (暖黄) | 注意、过渡 |
| 错误色 | #FF6B6B (柔红) | 高危、焦虑 |
| 考前夜特供 | #1A1A2E → #16213E | 深夜星空，安静不压抑 |

**时刻状态色**:
- 当前活跃: 主色渐变边框 + 呼吸光晕
- 已完成: 成功色 #00B894
- 下一个: 主色 #6C63FF 微亮
- 未解锁: 灰色 #B2BEC3

## 字体排版

- **中文**: System Default (PingFang SC / Noto Sans SC)
- **英文/数字**: System Default
- **大标题**: 28px, weight 800, 用于页面标题
- **副标题**: 18px, weight 700, 用于区块标题
- **正文**: 15px, weight 400, 用于描述
- **辅助文字**: 13px, weight 400, 用于标签、时间戳
- **数据大字**: 32px, weight 800, 用于关键数字（如 Band 分数）

## 动效与交互

- **呼吸光晕**: 当前时刻卡片的边框光晕，2s 循环，ease-in-out
- **时间线连接**: 从上到下的渐进显现动画
- **按压反馈**: scale 1 → 0.97, 100ms
- **页面转场**: slide_from_right（Stack 导航）
- **考前夜模式**: 缓慢的呼吸圆点动画（吸气4s → 屏住4s → 呼气6s）
- **缓动曲线**: ease-out 为主，给人"稳稳落地"的感觉

## 设计禁忌

- 不要使用冰冷的纯蓝色（#0000FF 类）- 这是情感支持工具，不是银行App
- 不要使用纯黑背景 - 即使是考前夜模式也用深蓝灰 (#1A1A2E)
- 不要用红色大面积渲染焦虑 - 用柔和的紫色和粉色传递"我理解你"
- 不要用生硬的直角 - 所有卡片至少 16px 圆角
- 不要用密集的信息排布 - 留白是安全感的一部分
- 不要在考前夜模式用任何明亮的颜色 - 只有柔和的月光白和深蓝
